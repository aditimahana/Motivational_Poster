# Motivational_Poster

**Overview**
- This project is a simple motivational poster website with a centred picture, a heading and a sentence at the bottom.
- This project motivates others with this poster and also inspires others to make motivational websites like these.

**What I used:**
- HTML: To create the structure and content of the website such as the text and the iamge
- CSS: To style everything by changing the background color, centering everything and more to make this website look visually appealling.

**Challenges I faced**
- Making everything in the centre
- Forgetting to make the text color white

**What I learned:**
- How to center everything
- Making margins to make the website look breahtable
- Adding an outline
